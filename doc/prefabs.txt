% Prefabs are done with a simple markup language.
%
% Comments are done with a single percent mark.
%
% Legend:
%    #   wall
%    •   lamp
%    .   floor
%    *   connection
%    ~   water
%    ≈   lava
%    ;   window
%    +   door
%    ±   locked door
%    ≡   bars
%    ?   anything
%    α   depends on level
%    β   depends on level
%    γ   depends on level
%    L   first loot item
%    X   second loot item
%    R   rare loot item
%
% Metadata:
%    :material <name>
%        Use <name> as the material for floor and wall tiles.
%        Reset on /.
%    :center_align
%        Disallow placing a subroom if it wouldn't be aligned in the center of
%        the parent room.
%
% Example:

?##*##?
##...##
#.....#
*.....*
#.....#
##...##
?##*##?
